GAVROCHE - Espace de travail Saint-Michel Capital
==================================================

INSTALLATION :
1. Renommez .env.example en .env
2. Remplacez METTEZ_VOTRE_CLE_ICI par votre clé API Anthropic
3. Dans l'invite de commandes, tapez : node server/index.js
4. Ouvrez votre navigateur sur : http://localhost:3000

FICHIERS :
- server/index.js  : le serveur
- public/          : les fichiers du site (HTML, photos)
- .env             : vos informations secrètes (ne pas partager)
